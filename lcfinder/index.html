LC's Finderv0.1 Beta

一个简单的图片管理工具

下载 Windows 应用商店

支持 Windows 7 及更高版本

LC's Finder 是一款图片资源管理工具，你可以用它来浏览图片、对图片进行简单的分类，或者搜索你想要的图片。它的功能不会仅仅只有这些，在未来会加入更多有趣实用的功能。

主要特性

轻量

与那些安装包动辄占用几十兆空间的软件不同，我们只专注图片管理，没有乱七八糟的功能。

干净

没有烦人的广告，没有捆绑软件，无需安装，下载解包即可使用，卸载也很方便，只需要直接删除文件夹。

开源

源代码完全公开，如果你想成为 LC's Finder 社区的一部分或想帮助改进它，可以在 GitHub 上找到我们。

标签与搜索

LC's Finder 的搜索功能是基于标签的，你只需要为图片贴上合适的标签，以后就能方便的搜索到它们。

多语言

作为面向普通用户的应用，考虑到用户的使用体验，我们加入了多语言支持，用户可以选择自己习惯的语言。

触控

与传统的桌面应用不同，我们加入了对触控的支持，习惯触屏操作的用户也能很方便的使用它。

未来计划

未来还有很多功能需要添加，如果你是一位开发者，看到以下待添加的功能中有一些是你熟悉并能够实现的，可以向我们提供技术支持。

图片编辑

只需要简单的图片编辑功能，包括：裁剪、旋转、缩放、马赛克、滤镜等。

GIF 动画

除了静态图片，相信还有很多人喜欢收藏一些有趣的 GIF 动图，要是不支持 GIF 动图那将是非常遗憾的事情。

跨平台

不仅限于 Windows 桌面平台，还需要添加对 Linux、Mac OS、iOS、Android 平台的支持。

智能整理

让用户手动为众多图片添加标签是一件麻烦事，要是能够识别图片内容并自动分类整理图片，那将为用户省去很多不必要的时间浪费。

优化性能

目前的界面在绘制时没有任何硬件加速，应用的窗口越大，界面的卡顿现象也会越明显，而且还存在内存泄漏的问题，这些都需要解决掉。

颜色搜索

有的时候需要找一张图片，但不知道具体内容是什么，只知道它大致包含的颜色，这时如果有颜色搜索功能的话，就能从颜色相近的图片中找了。
